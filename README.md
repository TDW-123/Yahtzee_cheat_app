# Yahtzee Smart Tracker Pro

## Omschrijving
Yahtzee Smart Tracker Pro is een C# Windows Forms applicatie ontwikkeld om spelers te ondersteunen bij het nemen van strategische beslissingen tijdens het spel Yahtzee. De applicatie analyseert de huidige worp en berekent in real-time de meest optimale zet, rekening houdend met de officiële spelregels.

## Kernfunctionaliteiten
- **Strategische Analyse**: De tool berekent de best mogelijke score voor de huidige worp op basis van de huidige status van het scoreformulier.
- **Yahtzee Bonus Logica**: Volledige implementatie van de 100-punten Yahtzee-bonus, inclusief de verplichte Joker-regels wanneer de Yahtzee-box reeds bezet is.
- **Upper Section Focus**: Automatische detectie of het behalen van de 35-punten bonus voor de bovenste helft (bij een totaal van 63+) haalbaar is.
- **Intuïtieve Gebruikersinterface**: Ontworpen voor snelheid en efficiëntie, zodat je tijdens het spelen direct het juiste advies krijgt.
- **Auto-invoer**: Gebruiksvriendelijke input-velden met automatische cursor-focus voor snelle invoer van dobbelsteenwaarden.

## Waarom Open Source?
Ik heb dit project open-source gemaakt omdat ik geloof in het delen van logische oplossingen voor strategische spellen. Door deze tool open-source te houden, hoop ik anderen in de community te helpen hun spelstrategie te verbeteren en biedt het ontwikkelaars een referentie voor het implementeren van complexe spelregels en logica in C#.

## Toekomstige Ontwikkelingen
- Uitbreiding met statistische kansberekeningen (waarschijnlijkheid van specifieke worpen).
- Support voor verschillende internationale Yahtzee-varianten.
- Verdere optimalisatie van de UI voor mobiele Windows-apparaten.
